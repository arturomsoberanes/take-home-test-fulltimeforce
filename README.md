Before starting :checkered_flag:, you need to have:

## :toolbox: Installation ##

Clone Repository

```bash
$ git clone https://github.com/arturomsoberanes/take-home-test-fulltimeforce.git
```
Go to directory

```bash
$ cd take-home-test-fulltimeforce
```
Install all dependencies.

```bash
$ npm install 
```

## :joystick:	How to Run ##

Access in directory

```bash
$ cd take-home-test-fulltimeforce
```

Execute "start"

```bash
$ npm start
```
